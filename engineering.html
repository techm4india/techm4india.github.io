Program Structure

Credit-Based
Semester Programs

Structured as semester-length programs aligned with university academic calendars — enabling seamless integration with B.Tech and M.Tech curriculum frameworks.

📐
🏗️ Semester I Program

Engineering Foundations & Digital Tools

Semester-1 introduction to industry software — CAD basics, BIM fundamentals, drawing standards, and engineering documentation practices.

  • AutoCAD 2D/3D — Drafting & Documentation
  • Revit Fundamentals — Architectural Modeling
  • Engineering Drawing Standards & BIS Codes
  • Industry workflow introduction & project brief
  • Capstone: Complete building layout drawing set
⚙️
⚙️ Semester II Program

Structural Analysis & Simulation

Advanced structural analysis workflows — from modeling to analysis to design verification using STAAD Pro, ETABS, and ANSYS.

  • STAAD Pro — Multi-storey structural modeling
  • ETABS — Seismic analysis & design
  • ANSYS — FEA simulation fundamentals
  • Load combinations, IS code compliance
  • Capstone: Complete structural analysis report
🔬
🔬 Advanced Program

Advanced Simulation & BIM Workflows

Industry-level advanced programs — HyperMesh meshing workflows, advanced BIM coordination, clash detection, and construction documentation.

  • HyperMesh — Advanced FE mesh generation
  • Revit Advanced — MEP & structural coordination
  • BIM 360 — Cloud collaboration workflows
  • Clash detection & model coordination
  • Capstone: Full BIM project delivery
